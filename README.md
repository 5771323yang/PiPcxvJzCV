## 核心功能
- 登录注册
- 填报指南
- 在线测评管理
- 模拟填报管理
- 院校信息
- 专业信息
- 分数线
- 我的收藏管理
- 用户管理
- 院校管理
- 专业管理
- 录取批次管理

- 模拟填报管理

## 适用场景
- 高考考生
- 教育咨询机构
- 高中学校
- 家长
